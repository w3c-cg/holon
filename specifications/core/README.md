# Core Architecture

Scope: the Holon vocabulary itself — what a holon is, how holons relate to
one another (`holon:parentHolon`, `holon:imports`, `holon:importsShapes`),
the root holon, and boundary/portal semantics.

This working group's output is expected to become the Holon Core Ontology
referenced in the [repository README](../../README.md).
